# cfetch
A simple system information tool for Linux.

![cfetch](https://raw.githubusercontent.com/clieg/clieg.github.io/master/images/cfetch-new.png)

##### Configuration
```
# Git clone cfetch
    git clone https://github.com/clieg/cfetch.git

# Go to cfetch's directory
    cd cfetch

# Install cfetch
    sudo make install

# Run cfetch
    cfetch

# cfetch usage
    cfetch --help
```
